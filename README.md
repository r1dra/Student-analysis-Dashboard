# 🎓 Student Performance Analysis

## 📌 Overview
This project analyzes student performance data using Excel, SQL, Python, and Power BI to uncover insights into academic trends and factors affecting student scores.

## 🛠️ Tools & Technologies
- Excel (Data Cleaning)
- SQL (Data Querying)
- Python (Data Analysis)
- Power BI (Dashboard Visualization)

## 📊 Key Features
- Data cleaning and preprocessing using Excel
- SQL queries to extract meaningful insights
- Python-based analysis (Pandas, Matplotlib)
- Interactive Power BI dashboard

## 📈 Insights
- Comparison of scores based on gender and ethnicity
- Impact of parental education on performance
- Subject-wise performance trends
